# 🎮 Tic Tac Toe with Smart AI & Two Player Mode

An interactive Tic Tac Toe game built with HTML, CSS, and JavaScript, supporting both **Two-Player Mode** and a challenging **AI Mode** with Minimax Algorithm. Includes dynamic win detection with a victory line animation. 🧠💥

---

## 🧠 Features

- 🔁 Restart Game Button  
- 🧠 AI Mode using Minimax Algorithm  
- 👥 2-Player mode support  
- 🏆 Victory line animation on win  
- 💻 Fully Responsive UI  
- 🎨 Stylish design with hover effects
